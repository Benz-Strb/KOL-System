1. KOL Profile Table (KOL_Profile_Table)
- KOL_ID : Unique identifier for the influencer (primary key, automatically generated by the system) 
- KOL Name / Handle : Influencer Name / Social Media Account Name 
- Platform: Affiliated platform (enumerated values: TikTok, Instagram, YouTube, Mercado Libre, etc.)
- Contact_Info: Contact Information (stores email, WhatsApp, etc., requires permission encryption)
- Audience_Tags: Audience Tags (automatically tagged by the system based on fan profiles)
- Custom_Tags: Custom Tags (Supports manual insertion of tags, user-defined)
- Main Selling Points: Main selling points (description of the product categories or styles that influencers are good at promoting)
2. Business and Financial Policy Table (KOL_Commercial_Terms_Table)
- Pricing_Type: Cooperation Model (enumerated values: Single Cooperation / Contracted Package / Pure Exchange / Commission, etc.)
- Single_Post_Price: Single Collaboration Price
- Package_Price: Contracted Package Price
- Multi_Platform_Price: Multi-platform Package Price
- Is_Barter: Whether to barter (Boolean value: True/False)
3. Performance and Sample Management Table (Sample_Management_Table)
- Sample_ID: Sample Tracking Number (Primary Key)
- Sample_Status: Sample Status (Enumerated values: To be shipped / Shipped / Signed for)
- Return_Policy: Return Policy (Enumerated values: Return Required / No Return Required)
- Associated_Brand: Partner Brand
- Associated_Product_SKU: Partner Product (Associated SKU Code, for easy subsequent filtering by product)
4. Dynamic Effect Tracking Table (KOL_Performance_Snapshot_Table)
- Campaign_ID: Campaign/Period ID
- Post_URL: Post/Video Jump Link (Supports Jump Link Tracking / Tracked Link)
- Tracking_Period: Statistical period (Daily daily crawl / Recent_30_Days aggregated over the past 30 days)
- Metrics_Impressions: Impressions
- Metrics_Likes: Number of Likes
- Metrics_Engagement_Rate: Engagement Rate (Calculation formula: $\frac{\text{Likes}+\text{Comments}+\text{Shares}}{\text{Total Exposure or Number of Followers}}$)
- Promotion_Status: Product promotion status (e.g., online, taken offline, hidden)
